This project demonstrates the fundamentals of unsupervised machine learning using a synthetic dataset. The dataset is generated with the make_blobs() function from Scikit-learn, which creates random data points grouped into clusters.

The goal of this project is to understand how machine learning models can identify patterns and group similar data points without labeled outputs. The dataset contains 1000 samples with 2 features and 3 cluster centers, allowing us to visualize cluster formation clearly.

Visualization of the dataset is performed using Matplotlib, helping to illustrate how clusters appear in two-dimensional space.
